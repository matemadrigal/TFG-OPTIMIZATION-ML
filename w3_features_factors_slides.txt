Missing Data

     Missing Data
         · Subject of a branch of Statistics
         · Let M be missing data indicator. Let  be parameters for joint
           distribution f of data Y, which is split into Yobs and Ymiss A
           missing data mechanism is
                 · MCAR - (independent of observed data)
                                        f(M.|Y, ) = f(M|) for all Ymiss, Yobs

                 · MAR - (condnl on obs data),
                                     f(M.|Y, ) = f(M|Yobs, ) for all Ymiss, Yobs

                 · MNAR - (unknown mech, systematic), M may depend on Ymiss (i.e.,
                   the values they take).

         · Related: Censored data, Rounded and Truncated

Multiple Imputation

     Multiple Imputation
     Multiple imputation involves three distinct phases. In Rubin's (1987)
     terminology:

        1. The missing values are filled in M times to generate M complete
           data sets.

        2. The M complete data sets are analyzed by using standard
           procedures.

        3. The results from the M analyses are combined / pooled into a
           single inference.

Problem with Outliers

     Outliers and Robust Statistics
         · In Robust Statistics, data is often characterized as being
           y  N(µ, 2) with probability p and y  N(µ, 10 × 2) with
           probability 1 - p.
         · Robust Statistics develops methods which are not sensitive to
           outliers, i.e.,
                 · use Trimmed-Mean or Median instead of mean,
                 · use MAD (median-absolute-deviation) instead of standard
                   deviation
                 · use Percentile Regression instead of OLS
         · A very large number of methods are not immune to outliers, so
           it is more common to identify and clean them

                                                                                                                                                   12
Outlier Detection

     Methods
         · Normal data - |y| > k  stdev(y) where everything is computed
           using previous values and k can be a large number say, 3 or 4.
         · Non normal data, interquartile range = IQR = 75th percentile
           -25th percentile.
         · Outliers if y > 75th percentile +3  IQR
         · Outliers if y < 25th percentile -3  IQR
         · Solutions:
                 · Replace by NaN - Use missing data method
                 · Remove observation
                 · Cap or Floor-Winsorization
                 · Categorisation

Definition: Asset Pricing Model
An asset pricing model is a quantitative model that describes the
value of a portfolio (including single-stock ones) based on a set of
financial indicators.

                                                                                                                                          30
CAPM, APT, Fama-French
Here we will briefly discuss three main asset pricing models:

   · Capital Asset Pricing Model (CAPM - Sharpe, Treynor, Lintner,
      Mossin, etc),

   · Arbitrage Pricing Theory (APT - Ross) A much more flexible
      framework

   · Fama-French a three/five/etc factor model. APT's most famous
      application

   · Each is the outcome of an equilibrium model where
      representative investors maximize the utility of their
      wealth/returns (i.e., MVO).

   · We will discuss in more detail when we discuss Portfolio
      Optimization.

CAPM
Capital Asset Pricing model (CAPM) describes the values of asset
returns by comparing against market returns,

   · In its most general formulation where ri is the return of the ith
      stock, rf is the risk-free return and rM is the market return
                            rit - rft = i + iM (rMt - rft) + it

   · Here  is a measure of the sensitivity of the stock to the market,
      with higher values expecting higher compensation for the risk

   · In this most general formulation  is not in the original
      formulation, and is a measure of the residual expected return of
      a portfolio (stock) return

Fama-French Model
Fama-French three factor model describes the values of portfolio
returns by considering the returns of

   · an average market portfolio,
   · a stock size factor (Small Minus Big factor, SMB), and
   · stock book-to-market ratio factor (High Minus Low factor, HML).
   · Mathematically, we have

              rit - rft = i + iM (rMt - rft) + isSMBt + ihHMLt + it
   · SMB and HML are in the original formulation, but many others

      are feasible

Fama-French Model Explanation

     Fama-French
     The model introduces multiple s,

         · the small-vs-big (SMB) factor, which weights a stock market
           capitalization,

         · the high-vs-low factor (HML), which weights the ratio between
           the fundamental valuation and the market valuation of a stock.

         · the market beta () which weights the overall sensitivity to
           market moves

         · Stocks characterised by high book-to-market ratios are called
           value stocks, while stocks characterised by low ratios
           correspond to growth stocks.

Factor Portfolios

     What is a factor?
         · Many other factors are possible
         · In Fama-French - a 'factor portfolio' was defined as a portfolio

Regularization

     What's going on?
         · Bias Variance Trade-off - Forecast Error
                          Forecast Error = (Bias)2 + Variance + noise
         · We can make this precise but essentially, we can reduce forecast
           error by reducing any of these terms (except ambient noise.
         · Low bias means we fit everything around (i.e., we get close to
           over-parameterised) and typically high variance
         · High bias may be under-parameterised, but the variance will
           typically be lower.

Futures Naming Convention

     Contract names usually 2/3 letter prefix:

           Sep:U, Oct:V, Nov:X, Dec:Z

     Finally comes year:
         · Usually 2 digits

     So, EDZ19 is Eurodollar Dec 2019, ESH20 is S&P500 Mini March 2020
